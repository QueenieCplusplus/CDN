# CDN
Content Delivery Network

存取層面：

（1）CDN 需要在這網路應用變化快速過程中，提升自己的可擴充性，同時讓系統保持良好的穩定性和容錯性（對劉服務的校正能力）。

（2）對內容的存取性進行安全性控管，支援使用者許可權的控制，可拒絕非法連線，並且支援防盜鏈機制。

（3）支援服務品質監控能力。

網路層面：

* 支援多點、分級、就近連線、multi-homed 拓樸架構。

應用層面：

1. 多螢幕設備：如電腦、手機、電視。

2. 支援單一傳播、多點傳輸的直撥。

3. 支援對時間及定位條的快速找出播放的點播。

4. 支援對電視頻道進行時間和節目上的錄製。
